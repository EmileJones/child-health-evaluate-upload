# 儿童健康评估系统自动上传脚本

本工具是 [child-health-evaluate](https://github.com/EmileJones/child-health-evaluate) 项目的配套上传脚本。它利用 Selenium 自动化技术，模拟人工操作，将存储在后台的儿童健康体检数据自动录入到 **国家基本公共卫生服务管理系统**。

## 🚀 项目背景

基层工作人员在进行儿童健康评估时，往往需要将大量纸质或本地系统的数据录入到国家指定的公共卫生平台。人工录入不仅工作量巨大、效率低下，而且由于数据繁杂，极易发生录入错误。本脚本通过对接 [child-health-evaluate](https://github.com/EmileJones/child-health-evaluate) 提供的接口，实现了数据的自动化流转。

## ✨ 主要功能

- **自动化录入**：支持 3 岁以下及 4-6 岁儿童健康检查记录的自动填报。
- **智能年龄适配**：根据儿童生日自动选择对应的录入表单（3 岁表单或 4-6 岁表单）。
- **数据项全覆盖**：包括身高、体重、视力（含 SPH、CYL、AXIS 等详细参数）、听力、牙齿、血红蛋白、发育评估及指导建议等。
- **异常处理与重试**：
  - 自动识别已录入的数据，避免重复提交。
  - 针对特定出生日期的儿童，支持跨年龄段重试录入。
  - 自动识别未建档儿童并进行统计。
- **实时统计**：运行结束后自动汇总录入成功、已存在、未建档及异常的人数。

## 🛠️ 技术栈

- **Python**: 核心逻辑实现。
- **Selenium**: 浏览器自动化控制。
- **Requests**: 与后端 API 进行数据交互。

## 📋 环境要求

1. **Python 3.10+**
2. **Chrome 浏览器**
3. **ChromeDriver**：请确保驱动版本与 Chrome 浏览器版本一致。
4. **后台服务**：需确保 [child-health-evaluate](https://github.com/EmileJones/child-health-evaluate) 后台已正常运行。

## ⚙️ 快速开始

### 1. 安装依赖
在项目根目录下运行：
```bash
pip install -r requirements.txt
```

### 2. 配置后台地址
本脚本默认连接的后台地址位于 `client/system/access/RequestServerAccess.py` 中：
```python
baseurl: str = "https://emilejones.top:3103/api/"
```
如需连接本地或其他服务器，请修改此处的 `baseurl`。

### 3. 运行程序
```bash
python main.py
```

### 4. 使用流程
1.  **输入运行时间**：程序启动后会提示“请输入期望运行时间(分)”，根据需求输入即可。
2.  **手动登录**：程序会自动打开 Chrome 浏览器并跳转至国家公共卫生管理系统登录页。**请在浏览器中手动完成登录操作**。
3.  **自动执行**：一旦用户进入系统主页，脚本将自动接管，开始从后台获取数据并执行录入。
4.  **查看报告**：程序运行结束或达到预设时间后，控制台会输出详细的录入统计报告。

## ⚠️ 注意事项

- **登录安全**：脚本不保存您的国家系统账号密码，登录过程需由人工在浏览器中完成，确保安全性。
- **网络稳定性**：录入过程中请保持网络畅通。如果网络波动导致页面加载超时，脚本会捕获异常并记录。
- **数据验证**：建议在脚本运行初期进行少量抽检，确保后台数据与录入项完全匹配。
